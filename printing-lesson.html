Printing is one of the most basic statements in Python.
It is extremely useful for terminal applications, to display information.
Here is an example on how it is used:


        print("Hello!")

Go ahead and input this into your IDE. The result should be that in the output box, it should say


        Hello!
To print variables, define a variable prior to the print statement and then use like this: 

        x = 5
        print(x)
This will output: 

        5
Have a go at this with different words and different values.
